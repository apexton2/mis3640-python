The Animal Name Goes Here

Create the functionality with JavaScript whereby when the user clicks on any of the four buttons, the image corresponding to that button is shown. Please write this as a single function. The same one function should be attached to each of the 4 buttons. Use "this.id" to write the function. This function will require a set of "if" statements to complete. Make sure you set the ALT and TITLE when you set the animal image. Change the heading of the page "The Animal Name Goes Here" to the animal type (Lynx, for e.g.) corresponding to the image shown.
